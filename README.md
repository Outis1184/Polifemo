## 🗎 How to Use 

### 1. File Preparation
The software is specifically designed to process .dat files generated by the Dinaten® (UGR) (https://core.ugr.es/dinaten/) system. 
This software is compatible with any Pendant Drop system file providing interfacial rheology data, as long as it contains the following headers (do not translate or rename):Image, Time(s), Tension(mN/m), and Area(mm²)
To ensure the automated analysis works correctly, your image entries must follow this naming convention:

**Pattern:** `Reo[DeltaArea%]-[Period]s[InternalID].img`

*   **Example:** `Reo30-50sReo9004.img`
    *   `30`: Area Delta expressed in percentage (%).
    *   `50s`: Oscillation period in seconds.
    *   `Reo9004`: Unique image identifier.

### 2. Execution
1.  Place the `AnalisisReologico.exe` file in the **same folder** where your `.dat` files are located.
2.  Double-click the executable.
3.  The program will automatically scan the directory, perform the harmonic analysis, and generate the results (plots and tables).

### 3. Requirements
- No Python installation required (Standalone version).
- Windows 10 or 11.
- Ensure you have read/write permissions in the folder to allow the software to save the output files.
